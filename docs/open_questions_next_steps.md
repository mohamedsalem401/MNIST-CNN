# Open Questions and Next-Step Research Directions

1. Hidden target quality
- Can local targets be replaced with learned or bi-level optimized targets to reduce heuristic bias?

2. Drift-aware memory
- Can memory values be made representation-invariant under encoder drift?

3. Retrieval reliability
- Can uncertainty-aware retrieval gating prevent harmful interventions?

4. Beyond MNIST
- Do effects persist on Fashion-MNIST, EMNIST, CIFAR-10, or long-tail datasets?

5. Memorization vs generalization
- How much gain remains under strict near-duplicate filtering and harder generalization tests?

6. Differentiable alternatives
- Can differentiable retrieval approximations improve optimization without destabilizing training?

7. Resource efficiency
- What is the Pareto frontier between accuracy gain, latency overhead, and memory footprint?
